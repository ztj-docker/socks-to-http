# docker-ztj1993-socks-to-http

## 相关链接
- [GitHub](https://github.com/ztj-docker/socks-to-http)

## 使用教程

> 简单使用

```
docker run -ti --rm ztj1993/socks-to-http -e HOST=127.0.0.1 -e PORT=1080
```
